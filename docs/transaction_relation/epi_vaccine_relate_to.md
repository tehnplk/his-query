# EPI_VACCINE Table Relationships (HOSxP)

This document lists related tables for columns in the `epi_vaccine` table within the `hos11253` database. In this database, `epi_vaccine` is a vaccine master table, not a transaction table.

## Common Query Pattern
```sql
SELECT
    e.epi_vaccine_id,
    e.epi_vaccine_name,
    e.vaccine_code,
    v.name AS vaccine_name,
    e.icode,
    d.name AS item_name,
    e.active_status
FROM epi_vaccine e
LEFT JOIN vaccine v ON v.code = e.vaccine_code
LEFT JOIN drugitems d ON d.icode = e.icode;
```

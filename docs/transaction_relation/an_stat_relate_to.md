## Common Query Pattern
```sql
SELECT
    a.an,
    a.hn,
    p.cid,
    a.regdate,
    a.dchdate,
    a.ward,
    w.name AS ward_name,
    a.pttype,
    pt.name AS pttype_name,
    a.pdx,
    icd.name AS pdx_name,
    a.rw,
    a.income
FROM an_stat a
LEFT JOIN patient p ON p.hn = a.hn
LEFT JOIN ward w ON w.ward = a.ward
LEFT JOIN pttype pt ON pt.pttype = a.pttype
LEFT JOIN icd101 icd ON icd.code = a.pdx
WHERE a.regdate BETWEEN '2026-02-01' AND '2026-02-29';
```

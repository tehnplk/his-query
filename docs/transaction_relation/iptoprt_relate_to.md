## Common Query Pattern
```sql
SELECT
    op.an,
    op.opdate,
    op.optime,
    op.enddate,
    op.endtime,
    op.icd9,
    icd.name AS icd9_name,
    op.doctor,
    d.name AS doctor_name,
    op.icode,
    op.iprice,
    op.iqty
FROM iptoprt op
LEFT JOIN icd9cm1 icd ON icd.code = op.icd9
LEFT JOIN doctor d ON d.code = op.doctor
WHERE op.an = '<AN>'
ORDER BY op.opdate, op.optime;
```

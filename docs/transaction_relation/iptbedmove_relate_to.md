## Common Query Pattern
```sql
SELECT
    m.an,
    m.movedate,
    m.movetime,
    m.oward,
    ow.name AS old_ward_name,
    m.obedno,
    m.nward,
    nw.name AS new_ward_name,
    m.nbedno,
    m.movereason
FROM iptbedmove m
LEFT JOIN ward ow ON ow.ward = m.oward
LEFT JOIN ward nw ON nw.ward = m.nward
WHERE m.an = '<AN>'
ORDER BY m.movedate, m.movetime;
```
